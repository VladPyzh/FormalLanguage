Пыж Владислав, задача 16.
Для решения данной задачи используется следующий алгоритм:
По регулярному выражению в польской записи мы строим НКА. Делается это просто поддерживая стек из НКА, за эту часть отвечает
функции parse_regex, update_stack, push_stack. Далее из НКА мы строим ДКА, удаляя пустые ребра, это метод класса Automat::DFA().
Получив ДКА, мы просто для каждой буквы исходного слова проходимся по максимально возможной длине из каждой вершины, из которой можно
дойти до терминальной. Таким образом мы просто рассмотрим все подслова, которые можно получить, используя данный автомат. 
Ассимптотика = O(построение ДКА) + O(размер ДКА) * O(длина слова).
